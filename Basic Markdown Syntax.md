
# Writing Markdown is Easy

###### Here are some useful syntaxes.
---
## Paragraphs
To create a heading, add # in front of a word or phrase.
Always put a space between the # and the heading name.
Put blank lines before and after a heading for compatibility.

# # Heading (< h1 > in HTML)

## ## Subheading (< h2 >)

### ### getting smaller (< h3 >)

###### ###### smallest (< h6 >)   
   



---

## Font

Add * before and after a word or phrase to make it *Italic*.

Add ** before and after a word or phrase to make it **Bold**.

Add *** before and after a word or phrase to make it ***Bold & Italic***.

Add ~~ before and after a word or phrase to have it ~~strikethrough~~.

Add == before and after a word or phrase to have it ==highlighted==.

---

## List

You can make a bulleted list by starting each line with a - followed by a space.

- Apple 🍎
- Orange 🍊
- Pineapple 🍍

You can make a numbered list by starting each line with a 1. followed by a space.

1. Sunny 🌞
2. Cloudy 🌥️
3. Rain 🌧️

You can make a nested section of a list by adding two spaces or ```tab``` before the bullet or number character.

- Apple 🍎
  - Apple pie 🥧
- Orange 🍊
- Pineapple 🍍

---

## Blockquotes

>Add > in front of a paragraph to create a blockquote.
>> Add >> or more to create nested blockquotes.

Put blank lines before and after blockquotes for compatibility.

---

## Code

Add ` before and after a code or command to create ``inline code``.

Add ``` before and after a code to create a code block.
```
Here is the code
```
---

## Table

Example of creating a table:

\| a \| b | c |
| - | - | - |
| 1 | 2 | 3 |
| 4 | 5 | 6 |

| a | b | c |
| - | - | - |
| 1 | 2 | 3 |
| 4 | 5 | 6 |

---

## Link & Images

Use `[Text](url)` to create an [External Link](url).

Insert image with `![alt text](img url)`.
